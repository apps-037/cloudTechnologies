AWS Certified Solutions Architect - Associate

Target Audience

The Course benefits both, students and experienced professionals, willing to achieve careers as Cloud Practitioner, Solutions Architect, Infrastructure Manager, Cloud Network Manager and Cloud Administrator.

Pre-Requisites

No Pre-requisite. However, any prior experience in Networking will be an advantage

Course Breakdown

  • Module 1: Introduction
    • Cloud Computing
    • AWS Global Infrastructure and its benefits
    • Creating an AWS Free Tier Account, Tracking Resource Usage, and Billing
  • Module 2: Networking Fundamentals
    • Introduction to Networking and IP Addressing
    • Subnetting and Classless Inter-Domain Routing (CIDR)
    • OSI Layers
    • TCP / IP Protocols
    • Network Security
    • Concept of NAT (Network Address Translation)
    • Public and Private IP Addresses
    • Network Utilities: Ping, TraceRoute
  • Modules 3: Elastic Compute Cloud (EC2)
    • EC2 Architecture
    • EC2 Storage
    • Security Groups
    • Bootstrap
    • AMI
    • Vertical Scaling
    • EBS Volume
    • EBS Snapshot
    • Placement Groups
  • Module 4: Virtual Private Cloud (VPC)
    • Route Tables
    • Internet Gateways
    • Elastic IP Addresses
    • Elastic Network Interfaces
    • Firewalls - Security Groups and Network Access Control Lists (ACLs)
    • NAT Instances
    • VPC Peering
  • Module 5: Elastic Load Balancing and Auto Scaling
    • Load Balancing Fundamentals
    • Classic Load Balancers
    • Network Load Balancers
    • Application Load Balancers
    • Launch Templates and Configurations
    • Auto Scaling Groups
  • Module 6: Simple Storage Service (S3) - Architecture and Features
    • Features of S3
    • Transferring Data to S3
    • Permissions - Bucket Policies and ACLs
    • Data Encryption
    • Hosting Static Website
    • Object Versioning
  • Module 7: Simple Storage Service (S3) - Performance and Resilience
    • S3 Storage Tiers/Classes
    • Lifecycle Management Policies
    • Cross-Region Replication (CRR)
  • Modules 8: Domain Name System (DNS) and Route 53
    • DNS Fundamentals
    • Domain Registration
    • Private vs Public Hosted Zones
    • DNS Record Types
    • Health Checks
    • Route 53 Fundamentals
  • Module 9: Amazon Database Services
    • Database overview
    • Relational Database Service (RDS - SQL Databases)
    • Amazon Redshift (Data Warehousing)
    • Amazon DynamoDB (NoSQL Databases)
  • Module 10: IAM (Identity and Access Management)
    • IAM Overview
    • IAM Users and Groups
    • IAM Roles
    • IAM Policies
  • Module 11: Containers and Kubernetes
    • Amazon Elastic Container Service (Amazon ECS)
    • Create ECS Cluster and Task
    • Create service for cluster to run task
    • Amazon Elastic Container Registry (ECR)
    • Deploy image in container from ECR Repository using ECS
    • What is Kubernetes and why we need it - Overview
    • Kubernetes Cluster Architecture
    • Provision an EKS Cluster
    • Deploy compute for EKS Cluster
    • Connect to EKS Cluster using Kubectl tool
    • AWS App Runner
    • AWS App Runner Deployment
  • Module 12: Application Integration
    • CloudWatch
    • CloudWatch Logs
  • Module 13: Logging and Monitoring
    • CloudWatch
    • CloudWatch Logs
  • Module 14: CloudFront (CDN - Content Delivery Network)
    • CloudFront Architecture
  • Module 15: Elasticache
    • In-Memory Caching
  • Module 16: Serverless Compute - Lambda
    • Lambda Essentials
