Target Audience

The Course benefits both, students and experienced professionals, willing to achieve careers as Cloud Practitioner, Solutions Architect, Infrastructure Manager, Cloud Network Manager and Cloud Administrator.

Pre-Requisites

No Pre-requisite. However, any prior experience in Networking will be an advantage

Course Breakdown

  • Module 1: Introduction
    • What is Cloud Computing
    • AWS Free Tier Account Creation Process
  • Module 2: AWS Datacenter Architecture
    • AWS Datacenter Architecture
    • AWS Regions & Availability Zone
  • Modules 3: Amazon Elastic Compute Cloud (Amazon EC2)
    • Windows EC2 Instance creation
    • Web site hosting on Windows EC2
    • Linux EC2 Instance Creation

    • On-Demand EC2 Instance
    • Reserved Instance
    • Scheduled Instance
    • Spot Instance
    • EC2 Template
    • Tenancy
  • Module 4: AWS Networking
    • Virtual Private Cloud (VPC)
    • NAT Gateways
    • VPC Peering
    • IP Addressing
    • Network Interface Configuration
    • VPN Client Endpoint
    • Open VPN
    • Site to Site VPN Connection
  • Module 5: Network Security
    • Security Group
    • Network ACL (NACL)
  • Module 6: AWS Command Line Interface (CLI)
    • AWS CLI configuration on Windows
    • AWS CLI Configuration on Linux
  • Module 7: Identity and Access Management (IAM)
    • Multi-Factor Authentication (MFA)
    • IAM - User & Group Creation
    • IAM Policy Creation - Using JSON
    • AWS Policy Generator
    • IAM - Concept of Role
  • Modules 8: AWS Data Management
    • Elastic Block Store - Volume Actions with Windows EC2
    • Elastic Block Store - Volume Actions with Linux EC2
    • EBS Volume Types
    • Elastic File System (EFS)
    • Ephemeral Storage or Instance Store
    • Amazon Simple Storage Service (S3)
    • S3 Bucket Versioning
    • Static Website Hosting on S3 Bucket
    • S3 Storage Class
    • S3 Life Cycle Rule
    • S3 Replication Rule
    • S3 Requester Pays
    • S3 Transfer Acceleration
    • S3 Bucket Mounting on Windows EC2 Instance
  • Module 9: High Availability
    • Elastic Load Balancer (ELB)
    • Domain Binding to ELB
    • Target Group Draining Mode
    • Load Balancing Settings
    • Auto Scaling Group
    • Auto Scaling Policies
  • Module 10: Monitoring and Metrics
    • Amazon Cloud Watch
    • Custom Metrics creation for Cloud Watch
    • Cloud Watch Events
    • Cloud Watch Billing Alarm
    • Cloud Watch Log Groups
    • Instance Status Checks
  • Module 11: Route 53
    • Route 53 & Domain Registration
    • Route 53 Hosted Zone
    • Create Route 53 record using alias
    • Route 53 Weighted Routing Policy
    • Route 53 Failover Routing Policy
    • Route 53 Geolocation Policy
    • Route 53 Latency Routing Policy
